KEY FEATURES
============

1. SCHEDULING STRATEGIES
   - FIFO: Orders processed in creation order
   - Priority: VIP orders prioritized over normal orders
   - SJF: Shortest Job First (fewer items = higher priority)
   - Dynamic switching with automatic queue reordering

2. REAL-TIME UPDATES
   - WebSocket (STOMP) for live kitchen dashboard updates
   - Polling fallback (5-second intervals) for queue refresh
   - SockJS support for older browsers

3. ORDER WORKFLOW
   CREATED ‚Üí PREPARING ‚Üí READY ‚Üí SERVED
   OR
   CREATED ‚Üí CANCELLED

4. TABLE MANAGEMENT
   AVAILABLE ‚Üí OCCUPIED ‚Üí CLEANING ‚Üí AVAILABLE
